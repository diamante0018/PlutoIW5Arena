# PlutoIW5Arena: A mod for Plutonium.pw

## Build

- Build a mod.ff archive using [OpenAssetTools][oat-link].
- Place the mod.ff archive in `%localappdata%/Plutonium/storage/iw5/mods/arena`.
- Set `fs_game` Dvar to `mods/arena`
